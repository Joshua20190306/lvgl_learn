# LVGL 学习路线指南

## 1. 前期准备

### 1.1 基础知识储备
- **C语言编程**：熟练掌握C语言基础知识，包括指针、结构体、函数指针等概念
- **嵌入式系统基础**：了解基本的嵌入式系统原理，包括GPIO、中断、定时器等概念
- **图形学基础**：了解基本的图形学概念，如像素、颜色深度、坐标系等

### 1.2 开发环境搭建
- **编译工具链**：安装GCC或Clang等C语言编译器
- **构建系统**：熟悉CMake的基本使用
- **版本控制**：掌握Git的基本操作
- **调试工具**：学会使用GDB等调试工具

## 2. LVGL基础入门

### 2.1 LVGL简介
- 了解LVGL是什么（Light and Versatile Graphics Library）
- 理解LVGL的特点和优势
- 掌握LVGL的应用场景

### 2.2 环境搭建与Hello World
- 下载并集成LVGL源码
- 配置lv_conf.h文件
- 创建第一个LVGL应用程序
- 在模拟器中运行（使用SDL2等后端）

### 2.3 LVGL核心概念
- 对象模型（Object Model）
- 显示驱动（Display Driver）
- 输入设备驱动（Input Device Driver）
- 事件系统（Event System）
- 样式系统（Style System）

## 3. 核心功能学习

### 3.1 基础控件
- **标签（Label）**：文本显示控件
- **按钮（Button）**：交互控件的基础
- **滑块（Slider）**：数值选择控件
- **开关（Switch）**：二进制选择控件
- **文本框（Text Area）**：文本输入控件
- **复选框（Check Box）**：多项选择控件
- **单选按钮（Radio Button）**：单项选择控件

### 3.2 布局系统
- Flex布局（Flex Layout）
- Grid布局（Grid Layout）
- 传统定位方法

### 3.3 样式与主题
- 样式属性详解
- 样式状态（normal, pressed, focused等）
- 主题定制
- 颜色系统与渐变

### 3.4 事件处理
- 事件类型与生命周期
- 事件回调函数
- 事件冒泡机制
- 自定义事件处理
